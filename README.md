# Rojima HP

## Rojimaの現状
- 出店希望申込はメール
- 出店希望の度にメールをしている
- 出店確認メールを送っている
- 出店者リストは現在なし
- Facebookで出店者紹介
- お越しの方へのページがない
- 最新情報は基本的にFacebookで更新している

## 要望
- ホームページから出店希望の申し込み
- 出店者の情報をDBで保持
- 出店者情報をCSVで出力
- メールアドレスとパスワードの認証により前回の入力情報が表示される
- 出店者リスト
- 簡単なブログ機能
